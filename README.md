# GTasksToolkit

Take control of your Google Tasks with this powerful desktop application designed for productivity enthusiasts and busy professionals. Available for Windows, macOS & Linux.

## Key Features:

- Connect securely to your Google Tasks account and download all your task lists
- Create, edit, update, and organize your tasks with an easy-to-use desktop interface
- Restore deleted Tasks
- Manage your tasks more efficiently than the web interface
- Instant search filtering and bulk editing
- Import Google Takeout JSON to migrate Tasks between accounts or restore backups
- Import/Export to spreadsheets/CSV (paid feature)

Install GTasksToolkit: https://www.gcaltoolkit.com/downloads/#gtaskstoolkit
